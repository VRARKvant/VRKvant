# Обзор Интерфейса Unity

## Окно Hierarchy (Иерархия)
![Окно иерархии](img/unity/interface/Hierarchy.png)
Это список всех объектов, которые находятся в вашей сцене прямо сейчас.

**Что здесь делать:**
* Создавать новые объекты (свет, кубы, персонажей).
* Выбирать объект, чтобы увидеть его свойства.
* Группировать объекты (перетаскивая один внутрь другого).
* Удалять объекты из сцены.

## Окно Scene (Сцена)
![Окно иерархии](img/unity/interface/Scene.png)
Это ваше **3D-окно** в игровой мир. Здесь вы видите всё «глазами разработчика».

**Что здесь делать:**
* Расставлять объекты в пространстве (двигать, крутить, масштабировать).
* Летать по миру и осматривать его с разных сторон.
* Выделять объекты визуально.

## Окно Game (Игра)
![Окно иерархии](img/unity/interface/Game.png)
Это окно показывает, как ваша игра будет выглядеть **на экране игрока** (через камеру).

**Что здесь делать:**
* Тестировать игру, когда нажата кнопка Play.
* Проверять, как выглядят цвета, свет и пользовательский интерфейс для игрока.
**Важно:** В этом окне нельзя двигать объекты вручную, оно только для просмотра.

## Окно Inspector (Инспектор)
![Окно иерархии](img/unity/interface/Inspector.png)
Это панель **настроек** выбранного объекта. Если в Иерархии вы выбрали объект, здесь появятся все его детали.

**Что здесь делать:**
* Менять координаты (Transform).
* Добавлять новые компоненты (например, физику — Rigidbody).
* Менять цвета, параметры скриптов и другие свойства.

## Окно Project (Проект)
![Окно иерархии](img/unity/interface/Project.png)
Это ваш **проводник файлов** (как папка на компьютере). Здесь хранятся все «исходники» вашей игры.

**Что здесь делать:**
* Хранить 3D-модели, картинки, звуки, скрипты и префабы.
* Перетаскивать файлы из этого окна в окно Scene, чтобы добавить их в игру.
* Организовывать файлы по папкам.
